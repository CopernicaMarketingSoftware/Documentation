# Magento quotes list

A Magento *quote* represents the content of a shopping cart. 
Theoretically the quote is an offer and if the user accepts it (by checking out) it converts to an order. 

The *Quotes* list provides targets that are quotes. Note that only quotes that 
have access to an email address are synchronized. 

## Personalization variables

Variables available for email personalization.

- [Magento](copernica-docs:MarketingSuite/magento-integration/object/magento) `$magento` 
- [Customer](copernica-docs:MarketingSuite/magento-integration/object/customer) `$customer`
- [Quote](copernica-docs:MarketingSuite/magento-integration/object/customer) `$quote`

## Filter options

Quotes list can have following filter options:

* **Active/inactive**
* **Customer Id**
* **Quantity**
* **Weight**
* **Total weight**
* **Chosen product**
* **Product's category**
* **Web store**
* **Currency**
* **IP address** 
* **Last modification date**
* **Finalized**
* **Empty/with items**
